# UNCP ACM Chapter Website

This is a web-based project built using PHP, HTML, CSS, and MySQL.

## Features
- User Registration & Login
- Member Dashboard
- Public Pages (Home, About, Events, Contact)
- Secure password hashing

## Setup Instructions
1. Import acm_db.sql into phpMyAdmin
2. Place project folder inside htdocs (XAMPP)
3. Run using localhost

This project was developed with some assistance from ChatGPT for debugging and understanding concepts.
